Structure recommandée

Une base simple à poser dans le dossier racine servi par ton webserv.

/www
├── index.html
├── features.html
├── upload.html
├── about.html
├── /assets
│   └── styles.css
└── /errors
    ├── 404.html
    └── 500.html

Racine

Mappe ton root serveur vers ce dossier.

Index

Utilise index.html comme page par défaut.

Errors

Référence /errors/404.html et /errors/500.html dans ta config.

Assets

Le CSS séparé permet de valider le service de fichiers statiques et les bons types MIME.
